何謂建構子 constructior

建構子 (Constructor) 是一種名稱和 Class 名稱相同，可以輸入參數，但沒有輸出值的特殊 method，在 class 被實體化的時候會被觸發，常被用來設定物件的初始值。

如何建立 class 的建構子

要設計一個類別的建構子，需要有以下要求:

  • method name 和 class name 需相同
  • 不需要設定回傳值型別
